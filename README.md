# RCLootCouncil History Viewer

A browser-based viewer compatible with [RCLootCouncil 2.0.3 for 3.3.5](https://github.com/andrew6180/RcLootCouncil-Wotlk) loot history. It runs entirely in your browser — no data is uploaded to any server.
